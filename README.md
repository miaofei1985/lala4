# Affiliate Transparency Board

A front-end dashboard for listing partner programs, pricing notes, payout models, regions, requirements, sources, and update dates.

## Run locally

```bash
npm install
npm run dev
```

## Main files

- `index.html`
- `src/main.js`
- `src/styles.css`
- `src/data/advertisers.js`

Prices change frequently. Replace sample rows with verified data from official dashboards, contracts, APIs, or CSV imports before production use.
